# wepy define plugin

## 安装

```bash
npm install @wepywu/plugin-define --save-dev
```

## 配置方式

**wepy.config.js**

```javascript
const DefinePlugin = require('@wepywu/plugin-define');

module.exports = {
  plugins: [
    DefinePlugin({
      BASE_URL: JSON.stringify('http://foobar.com'),
      'process.env.NODE_ENV': 'development',
      'typeof window': JSON.stringify('undefined'),
      DEV: true
    })
  ]
};
```
